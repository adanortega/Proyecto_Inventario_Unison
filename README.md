Proyecto de Reingeniería - Gestión de Inventarios
Sistema moderno desarrollado para la Universidad de Sonora (Tarea 14).

Tecnologías
Lenguaje: Java 17

Interfaz: JavaFX (MVC)

Persistencia: ORMLite con SQLite

Gestión: Maven

Características
Login seguro con cifrado MD5.

Gestión completa (CRUD) de Productos y Almacenes.

Relación automática entre stock y ubicación.

Interfaz oscura moderna y legible.
